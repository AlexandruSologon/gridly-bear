============
Installation
============


Stable release
--------------

To install grid-modelling-gui, run this command in your terminal:

.. code-block:: console

    $ pip install grid_modelling_gui

This is the preferred method to install grid-modelling-gui, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for grid-modelling-gui can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/asologon/grid_modelling_gui

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/asologon/grid_modelling_gui/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/asologon/grid_modelling_gui
.. _tarball: https://github.com/asologon/grid_modelling_gui/tarball/master
